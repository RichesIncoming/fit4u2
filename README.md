# Fitness Planner Streamlit App

## Setup

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Streamlit app:
```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`

## Features

- **Goal Selection**: Choose between lose, gain, or maintain weight
- **Personal Data**: Input weight (kg), height (cm), age, activity level, and diet preference
- **Auto-Calculation**: Calculates daily calories using the Mifflin-St Jeor equation
- **Personalized Plans**: Generates random workout routines and meal suggestions
- **Interactive UI**: User-friendly Streamlit interface with emoji indicators

## How It Works

1. Set your fitness goal and personal information
2. Click "Generate Plan"
3. View your personalized:
   - Daily caloric intake
   - 3-day workout schedule
   - Meal suggestions
